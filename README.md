[![Python Version](https://img.shields.io/badge/python-3.9%2B-blue)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-yellow)](LICENSE)

[![Code style: black](https://img.shields.io/badge/code%20style-black-000000.svg)](https://github.com/psf/black)

Приложение для преобразования изображений с фотоплёнок в отретушированные фотографии в домашних условиях **без использования фотосканера**. Достаточно обычной камеры или смартфона.

## ✨ Возможности

- 🎞️ **Оцифровка плёнки** — фотографируйте негативы или слайды на любой источник света
- 🖼️ **Автоматическая ретушь** — инверсия цветов, коррекция экспозиции, баланс белого
- 🔄 **Пакетная обработка** — обрабатывайте целые рулоны за раз
- 🧹 **Удаление пыли и царапин** — встроенные алгоритмы чистки изображений
- 📏 **Кадрирование** — автоматическое обнаружение границ кадра
- 🎨 **Профили под разные плёнки** — настройки для Kodak, Fujifilm, Ilford и других

## ⚙️ Установка и запуск

### Требования
- Python 3.9 или выше
- Камера или смартфон для съёмки плёнки (любой)

### Шаги по установке

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Artpardsur/photo_scaner.git
cd photo_scaner

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите приложение
python main.py
```

##📝 Использование

###Базовый рабочий процесс

```bash
# 1. Запустите приложение
python main.py

# 2. Загрузите фотографию негатива через интерфейс
#    (или используйте пакетный режим для папки)

# 3. Выберите тип плёнки из выпадающего списка

# 4. Нажмите «Обработать» — получите готовое фото

# 5. При необходимости отрегулируйте вручную:
#    - Экспозицию
#    - Контраст
#    - Насыщенность
#    - Ретушь пыли
```

###Консольный режим (для пакетной обработки)

```bash
# Обработка одного файла
python main.py --input negative.jpg --output photo.jpg --film kodak_gold

# Пакетная обработка папки
python main.py --input ./scans/ --output ./results/ --batch --auto-crop
```
